# Pattern Recognition Project

## Description

A Machine Learning project that predicts Titanic passenger survival using classification algorithms.

The project covers data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation. Multiple models including XGBoost, K-Nearest Neighbors (KNN), and Decision Tree were implemented and compared using Accuracy, Precision, Recall, F1-score, and ROC-AUC metrics.

This project demonstrates the application of machine learning techniques to analyze patterns in historical data and build predictive models.

## Project Files

- **Pattern_Recognition_Code.ipynb**  
  Contains the complete implementation, data analysis, model training, and evaluation process.

- **Report.pdf**  
  Contains the detailed project report and methodology.

- **Presentation.pptx**  
  Contains the project presentation slides.
